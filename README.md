## 📖 Overview

This project performs an end-to-end **Exploratory Data Analysis (EDA)** on an HR dataset using Python. It walks through the complete data analysis workflow — from cleaning and preprocessing to visualization and business storytelling — with the goal of uncovering employee trends that help organizations make smarter HR decisions.

<br>

## 🚀 Workflow

```
 Raw Dataset → Data Cleaning → Preprocessing → EDA → Visualization → Business Insights
```
